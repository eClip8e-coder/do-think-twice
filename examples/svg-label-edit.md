# SVG Label Edit

## User Prompt

Use do-think-twice to edit this SVG. Move the "Q2" label 10 px to the right. Do not change anything else.

## Without Do Think Twice

While editing the label, also improve the SVG by aligning all labels, changing the font weight, adjusting the chart colors, and cleaning up coordinates.

## With Do Think Twice

- Overall judgment: This is a narrow mechanical edit.
- Must fix: Increase only the `x` value for the "Q2" text element by 10.
- Strongly recommended: After the change, check whether the moved label overlaps a neighboring element.
- Optional: None.
- Not recommended: Do not restyle, recolor, resize, reformat, or realign unrelated SVG elements because the user explicitly constrained the edit.

## Why This Is Better

The guided response respects the user's constraint, reduces the chance of unintended visual changes, and still checks the only relevant risk introduced by the requested movement.
